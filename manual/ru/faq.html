Где fre:ac сохраняет свои файлы конфигурации?

fre:ac обычно сохраняет свою конфигурацию в файле freac.xml,который можно найти в папке fre:ac. Возможно, если fre:ac установлен в папке Program Files (обычно C:\Program Files или локализованный ее вариант), то она сохраняет свои файлы конфигурации в подпапке freac в папке Application Data.

Папкой Application Data обычно называют папку C:\Documents and Settings\<имя пользователя>\Application Data (или локализованный ее вариант) в Windows XP и C:\Users\<имя пользователя>\AppData\Roaming in Windows Vista.

Сохранение конфигурации в папке Application Data пользователя необходимо для поддержки различных конфигураций для многопользовательских систем.
