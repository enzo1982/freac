Var lagrar fre:ac sina konfigurationsfiler?

fre:ac lagrar som regel sin konfiguration i filen freac.xml, som hittas i fre:acs programkatalog. Om fre:ac emellertid är installerat i katalogen Program Files (vanligtvis C:\Program Files eller en lokaliserad version av densamma), lagrar det sina konfigurationsfiler i undermappen freac i katalogen Application Data.

Katalogen Application Data kallas vanligtvis C:\Documents and Settings\<user name>\Application Data (eller en lokaliserad version av densamma) i Windows XP och C:\Users\<user name>\AppData\Roaming i Windows Vista och senare.

Lagring av konfigurationen i användarens Application Data-katalog är nödvändigt för att stödja olika konfigurationer för flera användare på samma system.
