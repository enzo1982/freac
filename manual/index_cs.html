fre:ac • Uživatelská příručka

More language ... Příručka pro kopírování CD pro začátečníky i pokročilé uživatele

fre:ac je všestranný CD ripper, kodér a konvertor pro zvukové soubory. Podporuje formáty MP3, Ogg Vorbis a Wave. Můžete si stáhnout nejnovější verzi fre:ac na fre:ac-Homepage. fre:ac je Open Source program.

fre:ac umí kódovat vysoce kvalitní MP3 soubory pro Váš počítač nebo přenosný přehrávač.

Uživatelská příručka se neustále vyvíjí. Tato příručka je pro začátečníky. Návod pro zkušené uživatele není ještě hotov.
