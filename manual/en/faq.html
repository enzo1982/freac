Where does fre:ac store its configuration files?

fre:ac usually stores its configuration in the file freac.xml found in the fre:ac application directory. However, if fre:ac is installed in the Program Files directory (usually C:\Program Files or a localized version of that) it stores its configuration files in a freac subfolder of the Application Data directory.

The Application Data directory is usually named C:\Documents and Settings\<user name>\Application Data (or a localized version of that) in Windows XP and C:\Users\<user name>\AppData\Roaming in Windows Vista and later.

Storing the configuration in the user's Application Data directory is necessary to support different configurations for multiple users on the same system.
