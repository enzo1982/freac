¿Dónde guarda fre:ac sus archivos de configuración?

En Windows, fre:ac suele almacenar su configuración en el archivo freac.xml que se encuentra en el directorio de la aplicación fre:ac. Sin embargo, si fre:ac se instala en el directorio Archivos de programa (generalmente, C:\Ficheros de programa o una versión local del mismo) almacena sus archivos de configuración en una subcarpeta freac del directorio Datos de aplicación.

El directorio de Datos de Aplicación se llama normalmente C:\Documents and Settings\<nombre de usuario>\Datos de programa (o una versión local de esta ruta) en Windows XP y C:\Users\<nombre de usuario>\AppData\Roaming en Windows Vista y posteriores.

Es necesario guardar la configuración en el directorio de Datos de Aplicación de cada usuario para soportar diferentes configuraciones para múltiples usuarios del mismo sistema.
