Non biltegiratzen ditu fre:ac-ek bere itxurapen agiriak?

fre:ac-ek arrunt bere itxurapena freac.xml agirian biltegiratzen du, fre:ac aplikazioaren zuzenbidean gordetzen dena. Horrela ere, fre:ac Program Files zuzenbidean ezartzen bada (arrunt C:\Program Files edo bere bertsio lokalizatu batean) bere itxurapen agiriak Application Data zuzenbideko freac azpiagiritegi batean biltegiratzen ditu.

Application Data zuzenbidea arrun honela izendatzen da: C:\Documents and Settings\<erabiltzaile izena>\Application Data (edo bere bertsio lokalizatu batean) Windows XP-an eta C:\Users\<erabiltzaile izena>\AppData\Roaming Windows Vistan.

Itxurapena erabiltzailearen Application Data zuzenbidean biltegiratzea beharrezkoa da itxurapen ezberdinak erabiltzaile ezberdinentzat sostengatu ahal izateko sistema berdinean.
